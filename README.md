# 💳 AI-Powered Credit Card Fraud Detection System

## 🚀 Overview
This project is a Machine Learning-based Credit Card Fraud Detection system built using XGBoost and deployed with Streamlit.

The system:
- Detects fraudulent transactions
- Handles imbalanced dataset
- Provides fraud probability
- Shows analytics dashboard
- Displays model performance metrics

---

## 🧠 Tech Stack

- Python
- Streamlit
- XGBoost
- Scikit-learn
- Pandas
- Matplotlib & Seaborn

---

## 📊 Features

✔ Fraud Detection using XGBoost  
✔ Fraud Probability Output  
✔ Analytics Dashboard  
✔ Confusion Matrix  
✔ Class Distribution Visualization  
✔ Final Year Project Ready UI
